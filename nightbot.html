Nightbot Command Files

  • large arrays
  • + eyes.js :

    an array for a bunch of eye emotes we use. Array’s name is “eyes”

    variables: eyes

  • + gordonramsay.js :

    3 separate arrays “gr1”, “gr2”, “gr3” corresponding to @radravioli_’s insult generator

    variables: gr1, gr2, gr3

  • + LMS.js :

    [CONTAINS EXPLETIVES] an array with the transcript of the popular “lemme smash” meme.. Array’s name is “LMS”

    variables: LMS

  • + primes10k.js :

    A list of primes up to 10,000 in an array called “primes.”

    variables: primes

  • + rude.js :

    A list of rude words to use in conjuction with “testfor.js” below. As such, array name is “testFor”

    variables: testFor

  • + johnmulaney.js :

    [CONTAINS EXPLETIVES] various quotes by John Mulaney… but out of context in one array called “jm”.

    variables: jm

  • functions
  • + gaussian.js :

    creates a variable “gaus” with a normal distribution from (0,1) //originally made by joshuakcockrell on stackoverflow, adapted by gem

    variables: gaus

  • + typo_0.js :

    requires an input string “text” that will then be output with a random number of typos. Original iteration that swaps and duplicates some letters. No longer auto-sends message (returns “text” with typos)

    variables: words, text

  • + typo_1.js :

    same as the typo algo above but allows some control of level of typos made if the sentence contains “typo[number]”, up to 10.

    variables: text, words, iptNum, repeatNum

  • + typo_2.js :

    same as typo_1.js but with an improved function of also making typos with letters commonly mistyped on a QWERTY keyboard.

    variables: text, words, iptNum, repeatNum

  • + testFor.js :

    requires an input string “text” and array “testFor” that “text” will be tested against. Outputs “trg” that is true if triggered. Uses regex, case insensitive.

    variables: trg, text, testFor, regTest

  • + convert.js :

    Convert units to other units. Requires an input string ‘text’ in the form ‘[number] [inputUnit] [outputUnit]’. See the Reference Sheet for more details.

    variables: text, msg, val, helpTrg, cvrtvals, temperature, length, volume, massweight, area, time

  • + morse.js :

    Convert to and from morse code. Requires input string ‘text’. Morse code to word conversions must start with periods, dashes, or underscores. Spaces in morse should use _ or |.

    variables: text

  • specific commands
  • + fish_0.js :

    The original fish code! (with some added functionality to work with the current command) just change fish_1.js to fish_0.js to use the legacy code

    variables: fsh, msg, sz, fishoutput

  • + fish_1.js :

    2021, Feb 08 update: fish are now generated based on a normal distribution. Use this as the contents of the nightbot command

    variables: gaus, fsh, msg, sz, fishoutput

  • + fish_off.js :

    change fish_[x].js (x being version number) to fish_off.js to disable fishing without deleting the command

    variables: fishoutput

  • + fish_off_1.js :

    an alternative to fish_off.js based on Gabo’s code. Use fish_off_1.js to disable fishing with some quips.

    variables: fishoutput

  • for testing only
  • + test.js :

    this is just a js file with a const array “texttest”

    variables: testtext

  • + gaboemote.js :

    a long array of emotes Gabo sent me. array’s name is “emotes”

    variables: emotes
